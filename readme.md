# Django starter app

Complete setup using devcontainers.
Package include `tailwindcss` and `daisyUi`.

## Prerequisities
You need two things installed on your machine:

1. Docker
2. Visual Studio Code

## Running in development

- `npm run watch:css` 
- `python manage.py runserver`
